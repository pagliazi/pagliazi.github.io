CPP Learning

By Zayl on Nov 2, 2018

类的描述————声明 declaration

class CLASSNAME
{
private:
   data member declarations
public:
   member function prototypes
};//note semicolon at the end

C++将接口（类定义）放在.h头文件中，将实现（类方法）放在.cpp源代码文件中。

类的描述————实现

  • 定义类成员函数时，用作用域解析运算符（::）来标识函数所属的类
  • 类方法可以访问类的private组件
>

定义在类声明中的函数都自动转为内联函数；

根据改写规则，在类声明中定义方法等同于用原型替换方法定义，再在类声明的后面将定义改写成内联函数；

调用成员函数时，它将使用 调用它的 对象的数据成员

构造函数&析构函数

构造函数没有声明类型;
构造函数的参数表示的不是类成员而是赋给类成员的值


调用构造函数初始化对象：

Stock food = Stock("World", 25, 12.5); //显式调用
Stock food("World", 25, 12.5); //隐式调用

构造函数用于创建对象，但不能通过对象来调用

stock1.show(); //使用对象调用类方法

默认构造函数————在未提供显式初始值时，用来创建对象的构造函数

Stock fluffy_cat; //使用默认构造函数

若没有提供构造函数，C++会自动提供默认构造函数如：

Stock::stock() {}

只有在没定义任何构造函数的情况下，编译器才提供默认构造函数
要创建对象而不显式地初始化，则需定义一个不接受任何参数的默认构造函数
  • 给已有构造函数的参数赋默认值：
    • Stock(const string & co = "Error", int n = 0, double pr = 0.0);

  • 通过函数重载定义一个没有参数的构造函数
