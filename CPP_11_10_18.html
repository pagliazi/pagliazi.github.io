学习C++时讨论到类型问题

将int型指针地址转为double型指针地址

  • #include<iostream>
  • int main()
  • {
  • int* p=new int;
  • *p = 1；
  • double* d;
  • d = (double*)p;
  • std::cout << *d << std::endl;
  • }

将int型指针p地址的类型转为double型后，赋值给另一个指向double的指针d，两指针地址相同，但地址内存的值不同；因为d(double)多出4个字节存储空间，其中值未知

char型数组指针修改内存值效果

  • #include<iostream>
  • int main()
  • {
  • char* p=new char[4];
  • p[0]=10;
  • p[1]=2;
  • p[2]=0;
  • p[3]=0;
  • int* d;
  • d=(int*)p;
  • std::cout << *d << std::endl;
  • }

C++内存本质为char型，数组p将存储单元中的位数进行修改，p[0]=10，即低8位为0000 1010，p[1]=2，即第二个8位为0000 0010。

类型转换后0000 0010 0000 1010转为十进制数522。

浅拷贝类似于按址传递，深拷贝按值传递

浅拷贝

对象初始化对象的时候调动拷贝构造函数，只是拷贝指针指向的拷贝构造称为浅拷贝。

深拷贝

对象初始化对象的时候调动拷贝构造函数，真实开辟空间拷贝数据的拷贝构造称为深拷贝。

浅赋值

对象给对象赋值时候调动赋值语句，不会开辟空间，只是简单的指针赋值
