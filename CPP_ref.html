引用变量

By Zayl on Nov 6, 2018

引用变量是已定义变量的别名

创建引用变量

int &表示指向int类型的引用
声明引用变量时必须对其初始化

int & rodents = rats;
实际是下述代码的伪装表示：
int * const pr = rats;
 引用rodents 扮演的角色与 * pr 相同

将引用用作函数参数

使函数的变量名成为调用程序中变量的别名
按引用传递：允许被调用的函数能访问调用函数中的变量。
按值传递导致 被调用函数 使用 调用函数的值 的拷贝；
按值传递x创建新变量x，导致有2个变量，2个名称；
按引用传递&x使x成为传递来变量的别名，1个变量，2个名称。

临时变量、引用变量和 const

若实参和引用参数不匹配，将生成临时变量：

  • 实参类型正确，但不是左值；
  • 实参类型不正确，但可以转换为正确的类型。
左值参数是可被引用的数据对象，如变量、数组元素、结构成员、引用和解除引用的指针；
非左值包括字面常量和包含多项的表达式。
